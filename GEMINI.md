# GEMINI.md

Gemini CLI users should follow the same operating contract as other agents.

Load order:

1. `AGENTS.md`
2. `OCCUPATION_CONTRACT.md`
3. A role track from `tracks/`
4. A workflow mode from `modes/`
5. A user profile file such as `profile.yml`

Use Gemini for:

- profile audits
- role gap analysis
- GitHub profile improvement
- portfolio project planning
- interview preparation
- weekly execution plans

Do not generate fake achievements, fake metrics, fake testimonials, or
unreviewed application material.
