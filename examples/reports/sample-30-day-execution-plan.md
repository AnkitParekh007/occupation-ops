# Sample 30-Day Execution Plan

## Goal

Increase proof for AI Frontend Engineer applications before applying.

## Week 1

- rewrite GitHub profile README
- improve one flagship project README
- capture one screenshot or visual placeholder

## Week 2

- open three good-first issues on a flagship repo
- add one architecture note
- draft one interview story based on real work

## Week 3

- publish one technical update or repo walkthrough
- improve portfolio project descriptions
- tighten "mocked vs real" sections

## Week 4

- review public proof gaps again
- update pinned repos
- prepare an interview-safe proof summary

## Guardrail

Do not claim anything as shipped, used, or merged unless it is true and verifiable.
