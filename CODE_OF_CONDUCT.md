# Code of Conduct

## Our Pledge

We want Occupation-Ops to be a respectful, useful open-source project for
career-readiness tooling.

## Standards

Positive behavior:

- Be respectful and specific.
- Help improve truthful career artifacts.
- Label examples clearly.
- Protect private personal data.

Unacceptable behavior:

- Harassment or personal attacks.
- Publishing private information.
- Encouraging fake claims, fake metrics, or spam.
- Demanding unpaid personal career coaching from maintainers.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributions that
violate this code of conduct.
