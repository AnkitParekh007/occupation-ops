# Roadmap

## Near term

- strengthen the AI Frontend Architect flagship flow
- improve rubric-driven scoring
- expand validator coverage
- improve HTML dossier presentation
- keep JSON output parity with Markdown

## Medium term

- deepen Frontend Engineer and Full-Stack AI Engineer tracks
- add better sample bundles and screenshots
- tighten contributor extension points

## Deliberate non-goals for now

- browser automation
- job portal scanning
- mass application workflows
- recruiter CRM features

Those are not the current competitive wedge.
