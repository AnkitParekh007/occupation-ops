# Launch Checklist

## Repo

- CLI tests pass
- website typecheck passes
- website build passes
- generated website artifacts are not tracked in git
- sample profile and sample dossier are current

## Website

- homepage matches the current product wedge
- sample dossier page is live
- Quick Start is copy-paste verified
- CLI docs match the real command surface
- light and dark themes both render correctly

## Launch assets

- homepage screenshot ready
- dossier overview screenshot ready
- scorecard screenshot ready
- before/after proof asset ready
- short demo GIF ready

## LinkedIn launch

- one post draft focused on a concrete candidate problem
- one comment-ready sample screenshot
- one sample dossier link
- one clear CTA to try the repo or read the sample dossier
