# Profile Audit Mode

## Purpose

Audit a user's current career profile before they apply.

## Inputs

- Target occupation
- Resume summary
- GitHub profile link
- Portfolio link
- LinkedIn link
- Existing proof projects

## Output

- 30-second recruiter impression
- strongest proof
- weakest proof
- missing role signals
- unclear claims
- next five improvements

## Guardrails

Do not invent missing proof. Mark missing proof as TODO.
