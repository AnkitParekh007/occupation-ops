# GitHub Growth Mode

Improve a user's GitHub profile so it attracts recruiters, followers, forks, and
contribution requests without fake metrics or spam.

## Checklist

- Clear profile README.
- Strong pinned repos.
- Repo descriptions and topics.
- Screenshots or demo placeholders.
- CONTRIBUTING, ROADMAP, SECURITY, and issue templates.
- Good first issues.
- Honest "mocked vs real" sections.
