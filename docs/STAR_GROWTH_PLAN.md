# Star Growth Plan

## Principle

Earn stars by being useful, specific, and honest. No fake engagement, no
star-for-star, no spam.

## Week 1

- Launch with the AI Frontend Architect demo.
- Add 5 good-first issues.
- Share one post explaining the problem: become more hireable before applying.

## Week 2

- Add another occupation track example.
- Publish a sample profile audit.
- Ask for feedback from frontend and career-tech communities.

## Week 3

- Add JSON output to CLI scripts.
- Add tests.
- Create a screenshot of the generated report.

## Week 4

- Add one contributor-friendly roadmap item.
- Publish a short case study showing how to use the weekly plan.
- Invite maintainers to suggest new occupation tracks.
