# Career Coach Use Cases

Occupation-Ops can be useful to career coaches, mentors, and developer advocates who want a local, structured workflow for proof-oriented career improvement.

## Useful coaching scenarios

### 1. Role clarification

Help a client move from generic titles to a clearer target occupation with visible proof expectations.

### 2. Profile cleanup

Review GitHub, portfolio, and LinkedIn positioning for vague claims, missing proof, and weak public artifacts.

### 3. Project planning

Translate role gaps into one or two realistic proof projects rather than a vague "learn more AI" plan.

### 4. Interview preparation

Turn real projects into safer, more specific interview stories without exaggeration.

### 5. Weekly accountability

Use the weekly execution outputs to keep the client shipping visible proof instead of endlessly editing profile text.

## Why coaches may like this repo

- local-first
- easy-to-edit Markdown outputs
- reusable templates
- role-specific tracks
- strong truthfulness constraints

## Limits

This repo does not replace human coaching judgment. It works best as a structured workflow that a coach or mentor can adapt.
