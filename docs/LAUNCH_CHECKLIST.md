# Launch Checklist

## Repo Surface

- [ ] README explains the project in 30 seconds.
- [ ] Demo command works.
- [ ] Sample output exists.
- [ ] GitHub description and topics are updated.
- [ ] No inherited author story, metrics, screenshots, or links remain.
- [ ] No secrets or private career data are committed.

## MVP Quality

- [ ] `npm run doctor` passes.
- [ ] `npm run demo:ai-frontend` generates a report.
- [ ] `npm run audit:profile` prints a useful audit.
- [ ] `npm run plan:weekly` prints a weekly plan.
- [ ] Issue templates are relevant to Occupation-Ops.

## Growth

- [ ] Create 5 good-first issues.
- [ ] Add one screenshot of sample output.
- [ ] Add GitHub About metadata.
- [ ] Share launch post with honest MVP status.
