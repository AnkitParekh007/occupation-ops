# Roadmap

## v0.1.0 - CLI MVP (current)

- Rewrite README for Occupation-Ops.
- Add occupation contract, tracks, and modes.
- Add profile, resume, GitHub README, portfolio, weekly plan, and interview
  templates.
- Add AI Frontend Architect example workflow.
- Add `npm run init` — create profile.yml from template.
- Add `npm run audit:profile` — profile and positioning review.
- Add `npm run gap:role` — role gap analysis with 7-day improvement plan.
- Add `npm run portfolio:plan` — three portfolio project plans for AI frontend.
- Add `npm run github:growth` — GitHub profile, pinned repos, topics, issues.
- Add `npm run interview:stories` — STAR stories, system design, guardrails.
- Add `npm run plan:weekly` — weekly execution plan generator.
- Add `npm run demo:ai-frontend` — full AI Frontend Architect sample report.
- Add `npm run doctor` — repo health check with all commands listed.

## v0.2 - Better Scoring

- Add role-specific scoring rubrics.
- Add profile audit scorecards with numeric breakdowns.
- Add JSON output from scripts.
- Add more example profiles for non-frontend roles.

## v0.3 - Expanded Tracks and Modes

- Add occupation tracks for ML Engineer, Backend Engineer, and Tech Lead.
- Add repo README audit checklist mode.
- Add LinkedIn optimizer improvements.
- Add contribution-surface checklist.

## v0.4 - AI Provider Integration (optional)

- Add optional Claude or OpenAI integration for richer gap analysis output.
- Keep local-first mode as the default — API keys never required.
- Add prompt templates for custom model integrations.

## v0.5 - Interview System

- Add STAR story bank validator.
- Add role-specific interview topic maps.
- Add mock interview question sets.
- Add weekly interview prep plan generator.

## v1.0 - Stable Local Career OS

- Stable scripts with full test coverage.
- Complete docs for all tracks and modes.
- More occupation tracks across engineering, design, and product.
- Clear contribution process with good-first-issues.
- Optional lightweight local dashboard (no backend required).
