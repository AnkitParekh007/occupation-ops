# Frontend Engineer Track

## Goal

Show credible ability to build usable, accessible, maintainable web
applications.

## Proof Signals

- Clean portfolio with shipped or runnable projects.
- Strong README files with setup instructions.
- Responsive UI screenshots.
- Component structure and state management examples.
- Testing, accessibility, and performance notes.

## Project Ideas

- Dashboard with real state flows.
- Design system component gallery.
- Form-heavy workflow with validation and accessibility.

## Interview Topics

- Component boundaries.
- State management.
- Accessibility.
- Performance.
- API integration.
