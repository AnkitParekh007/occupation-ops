# AI Frontend Architect Track

## Target Role

AI Frontend Architect building enterprise Angular and TypeScript interfaces for
AI copilots, RAG workflows, MCP/tool execution, streaming UX, UI-aware agents,
approval flows, and enterprise-safe agentic systems.

## Proof Signals

- GitHub profile clearly states AI frontend positioning.
- At least three public proof repos demonstrate AI UI patterns.
- README files explain architecture, mocks, screenshots, and recruiter value.
- Portfolio includes project cards with problem, solution, stack, and takeaway.
- Resume ties frontend architecture experience to AI product workflows.
- Interview stories explain tradeoffs around streaming, RAG citations, tools,
  approvals, guardrails, and enterprise UX.
