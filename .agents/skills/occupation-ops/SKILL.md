# Occupation-Ops Skill

Use this skill when improving career-readiness workflows in Occupation-Ops.

## Purpose

Help users become more hireable before they apply by improving profile clarity,
role fit, public proof, portfolio plans, interview preparation, and weekly
execution.

## Rules

- Do not create fake claims.
- Do not invent metrics, testimonials, jobs, or merged contributions.
- Do not submit applications automatically.
- Keep personal data local.
- Label examples and mock outputs clearly.

## Workflow

1. Read `OCCUPATION_CONTRACT.md`.
2. Pick a track from `tracks/`.
3. Pick a mode from `modes/`.
4. Generate a concrete artifact from `templates/`.
5. Include next steps and truthfulness guardrails.
