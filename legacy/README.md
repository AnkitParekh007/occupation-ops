# Legacy Cleanup

Inherited job-application pipeline scripts and assets were removed from the
public MVP because they did not match the new Occupation-Ops positioning.

The launch MVP now focuses on:

- profile audits
- role-gap analysis
- GitHub growth
- portfolio planning
- interview preparation
- weekly execution plans

If any older scanner, PDF, tracker, or dashboard idea is reintroduced later, it
should be rebuilt around occupation readiness and documented as an
Occupation-Ops feature.
